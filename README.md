# EduShell OS

EduShell OS is an **sub operating system** that runs on top of **Windows**, designed specifically for **school ICT lab management**.  
It replaces the default Windows desktop with a controlled shell environment, helping educational institutions maintain a secure and distraction-free learning setup.

> ⚠️ **Project Status:** This project is currently under active development. **This project is not yet recommended for production use.**

---

## Overview

EduShell OS functions as a custom Windows shell.
Instead of allowing unrestricted access to the standard Windows interface, it provides a dedicated educational workspace where system access, applications, and user interactions can be controlled.

The primary goal of EduShell OS is to simplify computer lab management while ensuring students remain focused on academic tasks.

---

## Key Objectives

- Provide a controlled desktop environment for students  
- Reduce misuse of system features in school labs  
- Simplify ICT lab administration  
- Operate on existing Windows installations without requiring a full OS replacement

---

## Features (Planned & In Progress)

- Custom Electron-based desktop shell  
- Restricted access to Windows system features  
- Student-friendly and minimal user interface  
- Application control for lab environments  
- Designed for deployment in school ICT labs
